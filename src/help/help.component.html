About this app

This simple application is a partial proof-of-concept of a Progressive Web Application (PWA) for ABC Racing Company. Using service workers speeds up loading time, and supports off-line operation.

This application uses Firebase, a well-known Backend-as-a-Service (BaaS) system. Firebase provides hosting, authentication, S3-like storage, and database-in-the-cloud services. Its architecture supports completely real-time data updates.

The application uses Angular. To access Firebase, we use the wrapper known as AngularFire. The service worker support and other PWA support uses the relevant Angular PWA capabilities.

This application does not at present support server-side rendering. Server-side rendering is less important as a means of speeding up load time than it might otherwise be due to the use of Service Workers. However, the basic logic for SSR is present, and in fact simply running the relevant npm scripts, and then deploying the contents of the dist/server directory to a web server ought to work properly.
